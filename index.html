Projetos Acadêmicos

D-House móveis

O projeto para a loja de móveis fictícia foi desenvolvido utilizando HTML, CSS e JavaScript, O projeto conta com páginas dedicadas ao cadastro e login, além de uma seção para apresentação de produtos e um carrinho de compras. Adicionalmente, os usuários podem se inscrever na newsletter, garantindo acesso a novidades e promoções. A integração com o Google Maps permite que clientes encontrem facilmente a localização da loja, enquanto o carrossel de imagens enriquece a visualização dos itens disponíveis. Desenvolvido apenas com HTML, CSS e Javascript!

Notes

A aplicação desenvolvida em React, TypeScript e Tailwind permite a gravação de notas por meio da tecnologia de reconhecimento de fala da SpeechRecognition API, convertendo áudios em texto de forma eficiente. Além dessa funcionalidade, o sistema oferece opções para salvar notas digitadas, excluir registros indesejados e realizar buscas rápidas entre as anotações
